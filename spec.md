# 몰래몰레 (molemole) 기획 스펙

> 최종 업데이트: 2026-03-04

---

## 게임 개요

4×4 구멍 판에서 올라오는 두더지를 잡는 반응속도 게임.
매 턴 탄 수에 따라 두더지 + 스파이 등장. 일반 두더지만 클릭해야 함.

---

## 화면 흐름

```
인트로 (메인메뉴)  ← 스파이 두더지 1마리 / 버튼 4개 / 디버그 버튼
├── 게임 방법 → 게임 시작
├── 콜렉션
└── 설정 (미구현)

게임 중
├── 일시정지 → 계속하기 / 게임 중단(인트로로)
└── 게임 오버 → 다시 시작 / 메인메뉴
```

---

## 게임 규칙

| 항목 | 내용 |
|---|---|
| 그리드 | 4×4 (16칸) |
| 매 턴 등장 | 탄 수에 따라 변동 (아래 표 참고) |
| 클릭 목표 | 일반 두더지 1마리만 클릭 |
| 실패 조건 | 시간 초과 OR 스파이 클릭 |
| 점수 | 성공 턴 수 = 점수 |
| 등장 딜레이 | 이전 턴 성공 후 0.8~2초 랜덤 |

### 두더지 구성 (탄 수별)

| 탄 | 총 두더지 | 스파이 | 일반 |
|---|---|---|---|
| 0~9탄 | 3 | 1 | 2 |
| 10~19탄 | 4 | 2 | 2 |
| 20~29탄 | 4 | 3 | 1 |
| 30~39탄 | 5 | 3 | 2 |
| 40탄~ | 5 | 4 | 1 |

### 난이도 (제한시간)

| 점수 | 제한시간 |
|---|---|
| 0탄 | 2.0초 |
| 5탄 | 1.5초 |
| 10탄 | 1.0초 |
| 15탄 | 0.5초 |
| 20탄~ | 0.45초~ (최소 0.1초) |

## 무기 시스템

콜렉션에서 무기를 선택해 장착. 두더지 클릭 시 장착한 무기 이펙트 발동.

| 무기 | 이모지 | 상태 | 특징 |
|---|---|---|---|
| 뿅망치 | 🔨 | ✅ 완료 | 기본 무기, 망치 스윙 |
| 물총 | 🔫 | ✅ 완료 | 슬로우모션 + 물줄기 + 스플래시 |
| 번개 | ⚡ | ✅ 완료 | 번개가 두더지 머리에 맞음 |
| 폭탄 | 💣 | ✅ 완료 | 투척 이펙트 |
| 물풍선 | 🎈 | ✅ 완료 | 투척 이펙트 (폭탄 공용) |
| 핀조명 | 🔦 | ✅ 완료 | 램프에서 빔이 두더지로 뻗어나옴 |
| UFO빔 | 🛸 | ✅ 완료 | UFO 내려와서 두더지 납치, 위로 빨아올림 |
| 타겟 | 🎯 | ✅ 완료 | 조준선이 두더지를 향해 이동 |
| 인형뽑기 | 🦾 | ✅ 완료 | 갈고리가 내려와 두더지를 집어 위로 올라감 |

## 선물 시스템

| 단계 | 내용 |
|---|---|
| 등장 | 두더지 올라올 때 일반 두더지 칸에 🎁 같이 올라옴 (스파이 칸 없음) |
| 클릭 전 | 선물은 보이지만 클릭 불가 (두더지 뒤에 가려짐) |
| 두더지 처치 후 | 두더지 사라지고 선물만 남음 → 클릭 가능 |
| 선물 클릭 | 팝업에서 아이템 공개 |
| 수집하기 | 콜렉션에 아이템 추가 → 다음 턴 시작 |
| 드롭 풀 | 미획득 아이템 중 무기/모자/안경/장신구/효과 카테고리만 (테마/스킨 제외) |
| 중복 없음 | 이미 획득한 아이템은 재등장 안 함 |
