# 몰래몰레 (molemole) 기획 스펙

> 최종 업데이트: 2026-02-28

---

## 게임 개요

4×4 구멍 판에서 올라오는 두더지를 잡는 반응속도 게임.
매 턴 3칸에서 두더지 2마리 + 스파이 1마리 등장. 일반 두더지만 클릭해야 함.

---

## 화면 흐름

```
인트로 (메인메뉴)  ← 상단: 게임명 + 버튼 4개 / 하단: 두더지 4마리 행(스파이 1 포함)
├── 게임 방법 → 게임 시작
├── 콜렉션
└── 설정 (미구현)

게임 중
├── 일시정지 → 계속하기 / 게임 중단(인트로로)
└── 게임 오버 → 다시 시작 / 메인메뉴
```

---

## 게임 규칙

| 항목 | 내용 |
|---|---|
| 그리드 | 4×4 (16칸) |
| 매 턴 등장 | 일반 두더지 2 + 스파이 1 (랜덤 3칸) |
| 클릭 목표 | 일반 두더지 1마리만 클릭 |
| 실패 조건 | 시간 초과 OR 스파이 클릭 |
| 점수 | 성공 턴 수 = 점수 |
| 등장 딜레이 | 이전 턴 성공 후 2~5초 랜덤 |

### 난이도 (제한시간)

| 점수 | 제한시간 |
|---|---|
| 0탄 | 2.0초 |
| 5탄 | 1.5초 |
| 10탄 | 1.0초 |
| 15탄 | 0.5초 |
| 20탄~ | 0.45초~ (최소 0.1초) |

## 무기 시스템

콜렉션에서 무기를 선택해 장착. 두더지 클릭 시 장착한 무기 이펙트 발동.

| 무기 | 이모지 | 상태 | 특징 |
|---|---|---|---|
| 뿅망치 | 🔨 | ✅ 완료 | 기본 무기, 망치 스윙 |
| 물총 | 🔫 | ✅ 완료 | 슬로우모션 + 물줄기 + 스플래시 |
| 번개 | ⚡ | ✅ 완료 | 번개 이펙트 |
| 폭탄 | 💣 | ✅ 완료 | 투척 이펙트 |
| 물풍선 | 🎈 | ✅ 완료 | 투척 이펙트 (폭탄 공용) |
| 핀조명 | 🔦 | ✅ 완료 | 램프에서 빔이 두더지로 뻗어나옴 |
| UFO빔 | 🛸 | ✅ 완료 | UFO 내려와서 두더지 납치, 위로 빨아올림 |
| 타겟 | 🎯 | ✅ 완료 | 조준선이 두더지를 향해 이동 |
| 인형뽑기 | 🦾 | ✅ 완료 | 갈고리가 내려와 두더지를 집어 위로 올라감 |
| 그물 | 🪤 | ❌ 이펙트 미구현 | 현재 망치 fallback |
